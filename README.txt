【アプリについて】
一人暮らし向けの料理検索アプリ

【仕様】

■ログイン画面
・DBに格納されているユーザーID、パスワードでログインできる。
・ユーザーID、パスワードのいずれかでもDBの値と不一致であればエラーが発生する。
・ログイン処理が成功すると、料理検索画面に遷移する

■料理一覧画面
・料理検索画面で選択した食材、料理カテゴリと一致する料理名が一覧リストに表示される。
・検索条件がない場合は、全件取得される。
・料理名、食材、料理カテゴリが一覧として表示される。
・料理名をクリックすると、料理詳細画面に遷移する。

■料理詳細画面
・料理一覧画面でクリックした料理名の料理名、食材、料理カテゴリ、カロリー、料理イメージが詳細として表示される。
