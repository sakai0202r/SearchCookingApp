■概要
食材名・料理カテゴリを指定することで、条件と一致する料理名を検索することができるwebアプリです。
家に残っている食材と、その日の気分に合った料理カテゴリ(和食•洋食•中華など)で作りたい料理名が調べれたら便利かなと思い作成しました。
料理名検索だけではなく、ユーザー登録してお気に入りの料理を登録したり、カロリーやレシピの確認などを行うことが出来ます。

■使用技術

Java 1.8
java script
jquery 3.5.1
MySQL
Maven
jpa
